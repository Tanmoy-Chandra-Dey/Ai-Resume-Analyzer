# AI Resume Analyzer

An AI-powered resume analyzer that extracts skills, analyzes keywords, and calculates how well a resume matches a job description using Natural Language Processing (NLP).

This project helps job seekers understand whether their resume contains the right skills and keywords for a specific job.

---

## Features

- Extracts technical skills from resume text
- Matches resume keywords with job descriptions
- Calculates a resume-job match score
- Suggests improvements for better job matching
- Beginner-friendly AI/NLP project

---

## Technologies Used

- Python
- Natural Language Processing (NLP)
- Regular Expressions
